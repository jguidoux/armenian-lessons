# Leçon 0005 : sentiments et structure Ես [adj] եմ

## Vocabulaire introduit (6 mots)

| Arménien | Prononciation | Français |
|----------|--------------|---------|
| ուրախ | [ourakh] | content, joyeux |
| տխուր | [tekhour] | triste |
| հոգնած | [hognats] | fatigué(e) |
| լավ | [lav] | bien, bon |
| վատ | [vat] | mal, mauvais |
| սոված | [sovats] | affamé, j'ai faim |

## Point de grammaire : Ես + [adj] + եմ

Structure de base pour exprimer un état :
- **Ես ուրախ եմ** = Je suis content.
- **Ես լավ եմ** = Je vais bien.
- **Ես սոված եմ** = J'ai faim.

## Bonus découvert en session : la négation

Jérémie a produit spontanément **Սոված չեմ** = "Je ne suis pas affamé" sans avoir été enseigné la négation.

Règle : **չ** [ch] se préfixe au verbe être.

| Personne | Positif | Négatif |
|----------|---------|---------|
| 1re sg | եմ [em] | չեմ [chem] |
| 2e sg | ես [es] | չես [ches] |
| 3e sg | է [e] | չի [chi] |
