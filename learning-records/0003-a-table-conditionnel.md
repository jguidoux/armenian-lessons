# Leçon 0003 : vocabulaire de table et conditionnel de politesse

## Vocabulaire introduit (8 mots)

| Arménien | Prononciation | Français |
|----------|--------------|---------|
| թeyl | [tey] | thé |
| ջuр | [djour] | eau |
| հaц | [hats] | pain (aussi : nourriture) |
| կoφε | [kofé] | café |
| Бarі аkhorzak | [bari akhorjak] | Bon appétit |
| Шat hamov é | [shat hamov é] | C'est délicieux |
| Бavakan é | [bavakan é] | Ça suffit (refus poli) |
| кuzei | [kouzeyí] | je voudrais (conditionnel poli) |

## Point de grammaire : кuzei vs оuzum em

- **оuzum em** [ouzum em] = je veux — présent progressif, registre familier
- **кuzei** [kouzeyí] = je voudrais — conditionnel, registre poli ✓

Structure : **кuzei** + [aliment/boisson] (+ χndrem [khndrem] pour encore plus de politesse).

Ex. : кuzei тey, χndrem = Je voudrais du thé, s'il vous plaît.

## Formule de refus poli

**Шnorhakal em, бavakan é** [shnorhakal em, bavakan é] — Merci, ça suffit.

À répéter deux fois si la belle-mère insiste (coutume arménienne).

## Note culturelle

En Arménie, refuser au premier appel peut paraître impoli. La formule de refus doit être répétée. 
Dire **Бarі аkhorzak** répond à **Бarі аkhorzak** — on répond en miroir.
